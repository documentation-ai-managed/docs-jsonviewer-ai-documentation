## v2.1.0 (2024-10-15)

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time JSON validation with error highlighting directly in the tree view.
- Introduced export to CSV for tabular JSON data like arrays of objects.

## Improvements

- Optimized rendering for JSON files `{>10MB}` in size.
- Enhanced search functionality to match nested paths like `user.profile.address.street`.

## Bug Fixes

- Fixed collapse/expand states not persisting during validation errors.
- Resolved issue where beautified JSON lost comments in copy operations.

<CodeGroup tabs="Before,After">
```json
// Before: No error highlighting
{
  "users": [
    {
      "name": "Alice",
      "age": "invalid"  // No visual error
    }
  ]
}
```
```json
// After: Errors highlighted
{
  "users": [
    {
      "name": "Alice",
      "age": "invalid"  // Red underline + tooltip
    }
  ]
}
```
</CodeGroup>

</Update>

## v2.0.0 (2024-09-20)

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "improvement", "breaking"]}>

## New Features

- Overhauled interactive tree view with drag-to-reorder nodes.
- Added theme switcher supporting light, dark, and high-contrast modes.

## Improvements

- Increased beautify speed by `{40%}` for deeply nested structures.
- Better support for JSON Schema validation with custom schemas.

## Breaking Changes

- Removed legacy "compact" view mode; use beautify options instead.

## Bug Fixes

- Fixed parsing errors for JSON with trailing commas in objects.
- Corrected copy-to-clipboard for selected subtrees.

</Update>

## v1.9.0 (2024-08-10)

<Update label="2024-08-10" description="v1.9.0" tags={["improvement", "bugfix"]}>

## Improvements

- Accelerated load times for JSON payloads `{>5MB}` by lazy-loading branches.
- Added keyboard shortcuts: `<kbd>Ctrl</kbd>+<kbd>F</kbd>` for search, `<kbd>Ctrl</kbd>+<kbd>B</kbd>` to beautify.

## Bug Fixes

- Patched validation crash on malformed Unicode characters.
- Ensured consistent indentation in beautified output across browsers.
